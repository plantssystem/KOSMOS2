# 🎹 KOSMOS.sf2 – SoundFont 概要

## 1. KOSMOS.sf2 とは？

**KOSMOS.sf2** は、ジェネレーティブ MIDI シーケンサー *KOSMOS* の  
開発・テスト用途として作られた **軽量 SoundFont** です。

- メロディ生成の確認  
- ユークリッドリズムの検証  
- USB‑MIDI のタイミングチェック  

など、KOSMOS の動作を外部音源なしで即座に確認できるよう、  
**シンプルでニュートラルな音色**に設計されています。

---

## 2. 目的

KOSMOS.sf2 は以下の用途を想定して作られています：

- **MIDI 出力のテスト**  
- **スケール／モードの確認**（平調子・陰旋法など）  
- **アルペジオやフレーズ方向のデバッグ**  
- **クロック・タイミングの検証**  
- **モバイル／デスクトップ SoundFont プレイヤーでの即時再生**

CPU 負荷が非常に軽く、以下のようなアプリでの使用に最適です：

- bs‑16i  
- FluidSynth  
- SF2/SFZ 対応ソフトシンセ  
- Android / iOS の SoundFont プレイヤー

---

## 3. 音の特徴

- クリアで安定したトーン  
- リズムの明瞭さを重視した速いアタック  
- ニュートラルな倍音構成  
- 以下の用途に特に向く：
  - ユークリッドパルス  
  - アルペジオ  
  - 長尺のジェネレーティブフレーズ  
  - ベース／リードの分離確認  

目的は **「音色のリアルさ」ではなく「生成ロジックの可視化」** です。

## 5. 使い方

### **デスクトップ**
以下の SF2 対応ソフトで読み込み可能：

- FluidSynth  
- Polyphone  
- Sforzando（SFZ 経由）  
- MuseScore / LMMS / Qsynth  

### **モバイル**
推奨アプリ：

- **bs‑16i**（iOS / Android）  
- **Audio Evolution Mobile**  
- **SoundFont Pro**  

### **ハードウェア**
以下のような SF2 対応機器でも使用可能：

- Zynthian  
- MOD Devices  
- その他 SF2 対応サンプラー

## 8. 開発者向けメモ

- MIDI の挙動を明確にするため、あえて最小限の音色構成  
- 以下のテストに最適：
  - NoteOn/NoteOff のタイミング  
  - 確率ステップの発音確認  
  - フレーズ方向ロジック  
  - トランスポーズ挙動  
  - USB‑MIDI スループット  
- KOSMOS の内部ピッチレンジ（C2–C6）に最適化済み
