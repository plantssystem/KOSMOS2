# 🎹 KOSMOS MIDI RANDOMIZER for Renoise

**KOSMOS MIDI RANDOMIZER** は、KOSMOS2 から入力される MIDI ノートを  
**音楽的に意味のあるランダム性**へと変換する Renoise 用 Lua ツールです。

平調子（Hirajoshi）や陰音階（In Scale）などの  
**日本の伝統音階にも対応**しており、  
入力されたノートをリアルタイムでスケールに沿って再マッピングします。

これにより、KOSMOS2 のジェネレーティブフレーズに  
さらにもう一段階の **「揺らぎ」** と **「音楽的ランダム性」** を加えることができます。

---

## ✨ 特徴

- **リアルタイム MIDI ランダマイズ**  
  KOSMOS2 や MIDI キーボードからのノートを即座に変換。

- **日本音階に対応（平調子・陰音階など）**  
  独特の情緒を持つ自然なメロディを生成。

- **音楽的ランダム性のコントロール**  
  ピッチ、オクターブ範囲、ベロシティ、発音確率などを GUI で調整可能。

- **Renoise ネイティブで高速動作**  
  軽量 Lua 実装で低レイテンシ。

- **KOSMOS2 との相性抜群**  
  ハードウェア生成 × Renoise ランダマイズ × DAW エフェクト  
  → 三層構造の強力なジェネレーティブ音楽エンジン。
