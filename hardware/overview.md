# KOSMOS – ハードウェア概要  
RP2040 をベースにしたコンパクトなジェネレーティブ MIDI シーケンサー

---

## 1. ハードウェア概要

KOSMOS は **Raspberry Pi Pico（RP2040）** と  
**240×240 SPI LCD** を中心に構成された  
**スタンドアロン型ジェネレーティブ MIDI シーケンサー**です。

設計の特徴：

- 配線が最小限  
- 低消費電力  
- 電源投入後すぐに動作  
- USB‑MIDI デバイスとして使用可能  

構成部品：

- Raspberry Pi Pico（RP2040）  
- Waveshare Pico‑LCD‑1.3（ST7789）  
- デジタルボタン ×4（A/B/X/Y）  
- アナログジョイスティック（X/Y + スイッチ）  
- オプション：Pico Omnibus / デュアルエキスパンダ  
- USB‑MIDI（Pico の USB ポート経由）

外部クロック・外部ストレージ・外部オーディオは不要です。

---

## 2. 配線図（テキスト版）

### **LCD（Waveshare Pico‑LCD‑1.3）**  
**SPI1** に接続：

| LCD 信号 | Pico ピン | 説明 |
|----------|-----------|------|
| DC       | GP8       | Data/Command |
| CS       | GP9       | チップセレクト |
| RST      | GP12      | リセット |
| BL       | GP13      | バックライト |
| SCK      | GP10      | SPI1 SCK |
| MOSI     | GP11      | SPI1 TX |
| MISO     | —         | 使用しない |

電源は Pico の **3.3V / GND** から供給します。

---

### **ボタン（A/B/X/Y）**

各ボタンは：

- 一方 → **Pico GPIO**  
- もう一方 → **GND**  
- ファームウェア側は **INPUT_PULLUP** を使用

| ボタン | Pico ピン |
|--------|-----------|
| A      | GP15 |
| B      | GP17 |
| X      | GP19 |
| Y      | GP21 |

---

## 3. 電源と USB

- **USB‑C / USB‑micro**（Pico の種類による）で給電  
- USB 接続で以下を提供：
  - 電源  
  - USB‑MIDI デバイス機能  
- 外部 5V レギュレータ不要  
- 消費電流：**約 40〜70mA**（LCD バックライト ON 時）

---

## 4. オプションハードウェア

### **Pico Omnibus / デュアルエキスパンダ**

おすすめ理由：

- GPIO へのアクセスが容易  
- 配線が綺麗にまとまる  
- LCD + ジョイスティック + ボタンの安定した固定が可能

### **筐体（エンクロージャ）**

KOSMOS は以下のような筐体に組み込めます：

- 3D プリントケース  
- レーザーカットアクリル  
- ブレッドボード試作機  
- Eurorack 風パネル（USB 給電）

---

## 5. 推奨レイアウト（例）

```
 ┌───────────────────────────────┐
 │      240×240 LCD（メインUI）  │
 ├───────────────────────────────┤
 │   A    B    X    Y   （ボタン）│
 │                                 │
 │       ジョイスティック         │
 │     （X/Y + スイッチ）         │
 └───────────────────────────────┘
```

LCD がメイン UI、  
ボタンとジョイスティックがパラメータ操作を担当します。

---

## 6. 電気的注意点

- すべての GPIO は **3.3V 専用**（RP2040 の仕様）  
- レベルシフタ不要  
- LCD バックライトは GP13 から直接駆動  
- SPI1 は **24MHz** で駆動し高速描画  
- ボタンは内部プルアップを使用（外部抵抗不要）  
- ジョイスティックは安定した 3.3V 参照が必要（ADC 精度のため）

---

## 7. 最小構成の部品リスト（BOM）

| 部品 | 数量 | 備考 |
|------|------|------|
| Raspberry Pi Pico | 1 | RP2040 |
| Waveshare Pico‑LCD‑1.3 | 1 | ST7789 |
| タクトスイッチ | 4 | A/B/X/Y |
| アナログジョイスティック | 1 | X/Y + SW |
| 配線 / ピンヘッダ | — | はんだ or ジャンパー |
| USB ケーブル | 1 | 電源 + MIDI |

---

## 8. 組み立て手順（簡易）

1. Pico にピンヘッダをはんだ付け  
2. Omnibus に Pico を装着（任意）  
3. LCD を SPI1 ピンに接続  
4. ボタンを GPIO + GND に接続  
5. ジョイスティックを GP27/26/3 に接続  
6. USB で給電  
7. UF2 を書き込む → KOSMOS が即起動
